# Kalkulator
Prosty kalkulator konsolowy stworzony na potrzeby projektu zaliczeniowego.

## Funkcjonalności
- Dodawanie
- Odejmowanie
- Mnożenie
- Dzielenie
  
## Instalacja
```bash
# Klonowanie repozytorium
git clone [https://github.com/sped17/Kalkulator]
# Przejście do katalogu projektu
cd repozytorium
# Instalacja zależności
npm install
```
